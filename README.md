# XO ❌⭕️

A simple browser game where you play the classic XO (Tic-Tac-Toe) against a bot with 3 difficulty
levels.

## 🎯 Features

- Easy, Medium, and Hard bot modes
- Win and tie detection
- Clean UI and instant reset

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
